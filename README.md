## Installation

Note: You need have `Node` and `truffle` installed.

After have downloaded the repo, go inside with a terminal and start truffle.

```
truffle develop
> compile
> test
> migrate --reset
> test
```

And with a second terminal, open the client (dapp front-end).

```
cd client
npm install
npm test
npm start
```
